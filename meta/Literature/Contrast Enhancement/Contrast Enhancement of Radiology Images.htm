Contrast enhancement of radiology images

January 9, 2014Letters on digital image processingcontrast enhancement, contrast stretching, histogram equalization, radiology, x-rayrosmarus

Letters on Digital Image Processing, Case studies, Study 1.1

Due to the physical properties of the diagnostic equipment that radiology is using today, most of the images are suffering from low contrast. This is true for X-ray images as well and in this case study we are going to show you an unconventional trick of how to enhance low contrast X-Ray images without loosing details or adding any artifact.

1.1.1 Conventional contrast enhancement techniques

Within the set of digital image processing algorithms we have many techniques to enhance the contrast of a certain image. Just to mention the most important ones we have local and global operators or histogram based techniques.

Global arithmetic operators

By simply multiplying every pixel value with a given number you can gain contrast enhancement.

Contrast stretching

You can redefine the range of pixel intensities by stretching selected regions or compressing a certain band to a lower scale.

Histogram equalization

Automatically define a monotonically increasing discrete transfer function using the histogram data for a better use of the whole range of intensity values.

For a given image one of these techniques can give better result than the other, or a combination of these might lead to the desired solution. On top of theses techniques using the image to correct itself one can find even better enhancement for certain images, as show in the next chapter.

1.1.2 The self compensating technique

One can achieve pretty good result by using the image itself to compensate the low contrast regions. Once you know the uneven, in-homogeneous pattern of contrast and illumination changes of an image then you can compensate back using this information. The good news is that the image itself contains this information, we only have to use it.

TIP: The image itself always contains the pattern where the brightness is in-homogeneous or contrast enhancement would needed.

In our example we are going to do it manually but you can find an algorithm to automatize this process. Let’s take a look at the image in Fig. 1.1.1.

Fig. 1.1.1 Radiograph of human hips. Original image.

One can see that this image is not optimal. The upper region is much brighter while the hip joints and below are darker. The whole image contrast is low even if it considered to be a good X-ray image.

Fig. 1.1.2 Histogram of Fig. 1.1.1

As you can see in it’s histogram Fig. 1.1.2 not the whole pixel value range is in use, just pat of it. We will fix this by using contrast stretching in Walrus Vision Toolbox.

Fig. 1.1.3 Contrast stretching of  Fig. 1.1.1 using Walrus Vision Toolbox.

While stretching intensity regions the contrast of the bones in the image become much better. More details are visible for a human observer while using the whole range of pixel values.

Stretching the contrast either manually or using histogram equalization still not going to solve the problem of uneven brightness of the image. To compensate this we are going to use the negative of the contrast enhanced image as show in Fig. 1.1.4.

Fig. 1.1.4 The negative of the contrast stretched image.

 The negative of the image is suffering from the same uneven brightness but where the original is brighter than needed the negative is darker than needed.

TIP: Where the original image is brighter than needed, it’s negative is darker than needed. This statement is trivial but easy to forget and easy not to take the advantage of it.

We will amplify this difference by combining the two images as show in Fig. 1.1.5.

 Although this method provides particularity good results in highlighting bone structures and smaller details there is one problem with it. In radiology the brightness of the image holds the information about the density of the material that the X-rays penetrating through. In our false colored contrast image this information is lost or corrupted.

1.1.3 Highlight the density information

In order to highlight the density we can simply use colors. Colors are powerful descriptors while human eye can distinguish them better than shade of grays. We are going to use the image in Fig. 1.1.8 what is simply contrast stretched version of the original image.

Fig. 1.1.8 The original image after contrast stretching.

Fig. 1.1.9 False colored image using the HOT color tone filter. Below you can see the mapping of the different intensity levels.

Fig. 1.1.10 False colored image using the SPECTRUM color tone filter.  Below you can see the mapping of the different intensity levels.

In Fig. 1.1.9 and Fig 1.1.10 we have been using a different mapping of the intensity values than black to white. As when we have applied to SPECTRUM color tone filter the bright regions appeared with reddish color while the darker ones as blue.

If we put all these information together we will end up with Fig.1.1.11.

Intensity Extension Filter Set

December 11, 2013Walrus Vision Toolbox User Manualcontrast stretching, intensity adjustment, nonlinear transfer function, RGBrosmarus

Walrus Vision Toolbox User Manual Section 11.4 Plug-in description

Intensity adjustments of digital images are important to compensate illumination in-homogeneity between multiple frames, to readjust an image taken with improper exposure settings, to highlight or depress certain image regions or to modify high dynamic range images. The list of applications could be much longer. That is the reason why Walrus Vision Toolbox has an intensity extension filter set for digital image adjustments.

11.4.1 Intensity Extension Filter Set

In this filter set we have packed features to apply global intensity operations on your digital images. Although you have the most common intensity adjustment algorithms in the free version this filter set provides more control options, channel wise operations and nonlinear characteristics.

This filter set contains the following filters:

  • RGB Contrast Stretching Linear
  • RGB Contrast Stretching Nonlinear
  • Global Intensity Operation
  • Intensity Band
11.4.2 RGB Contrast Stretching Linear

With this filter you can apply channel wise linear contrast stretching on your digital images. You can re-adjust the red, green and blue channels intensity using a new linear mapping to each of the channels. The three transfer functions are completely independent from each other.

Fig. 11.4.1. Channel wise linear contrast stretching in action. On top left corner you can see the original image on top of it’s histogram.At the center the image after manual channel contrast stretching.

As you can see in Fig. 11.4.1 you can adjust the color channels manually to define a new intensity transfer function to the red, green and blue colors. The original image had too reddish color as you can see on the original histogram also. By readjusting the channels manually we could reduce the red dominance and  we gain slightly more realistic colors.

Just drag the control points under the filter properties and the image is updated automatically by re-scaled with the new transfer functions.

11.4.3 RGB Contrast Stretching Nonlinear

You have the option to define nonlinear transfer functions as well by using the channel wise nonlinear contrast stretching.

Fig. 11.4.2.  Channel wise nonlinear contrast stretching in action.

With the nonlinear adjustment option you can define fine tuned mixture of your color channels and reach special results in your intensity adjustments.

Just drag the control points and define the nonlinear transfer functions to tune your image intensity the way like.

11.4.4 Global Intensity Operation

You may want to apply the same transformation on all of your pixels like add a given value or apply a logical function. With the global intensity operation you can do that. You can choose from the following operations:

  • ADD
  • SUBTRACT
  • MULTIPLY
  • DIVIDE
  • AND
  • OR
  • XOR

As shown on Fig. 11.4.3 you can also select the channels you may want to apply the operation on.

Fig. 11.4.3. The properties panel of the global intensity operation filter.

11.4.5 Intensity Band

This filter is useful when you may want to keep only an intensity band from your full range image. You can set the lower and upper threshold points by dragging the control points up to the proper position. All pixel values below the lower threshold will be set to the min value and all pixel values above the upper threshold will be set to the max value.

 Fig. 11.4.4. The properties panel of the intensity band filter.

You can apply the narrowed intensity band to all of the channels or you can leave the not selected ones unchanged.
